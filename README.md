# Prerequisites
git  
bash  
vim  
eslint installed in the project  
prettier installed in the project  

# Getting started
## Installing
`git clone --recurse-submodules https://github.com/mdatsev/vim-config-bundle/`  
`cd vim-config-bundle`  
`./install.sh`
## Updating
Just run `./install` again

# Default keys
Use in normal mode  
F2: toggle linter  
F3: toggle sidebar  
F4: fix with linter  
F5: prettify  
F6: fix and prettify  
F7: relint  
F8: go to previous error  
F9: go to next error
